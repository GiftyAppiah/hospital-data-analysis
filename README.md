

## **Hospital Data Analysis (Excel & Power BI)**

This project analyzes hospital data to identify trends in patient visits, treatment costs, length of stay, payment status, and satisfaction levels. It helps support better hospital planning, decision-making, and service improvement.

---

### **Tools Used**

* **Microsoft Excel** – Data cleaning, pivot tables, initial analysis
* **Power BI** – Interactive dashboard and visual reporting





---

### **Key Focus Areas**

* Patient demographics and visit patterns
* Treatment cost analysis
* Length of hospital stay
* Payment status trends
* Patient satisfaction levels

 **Goal**
